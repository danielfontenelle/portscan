Necessário instalar o pacote do nmap.
   Tutorial:
   No Linux, ubuntu, etc...: sudo apt-get install python-nmap 
   No Windows ou linux que utiliza o pip: pip install python-nmap
Script criado em Python, com ele você vai ter maiores facilidades para fazer escaneamento de portas e fugir dos comandos utilizados no   NMAP. 


Comando para executar o script: python PortScanner.py
